# 7세그먼트: 공통 애노드(Common Anode) vs 공통 캐소드(Common Cathode)

7세그먼트 디스플레이는 여러 개의 LED로 구성되어 있으며, 이 LED들을 효율적으로 제어하기 위해 공통 단자로 묶는 방식에 따라 '공통 애노드'와 '공통 캐소드'로 나뉩니다. 두 방식은 제어 신호의 논리가 정반대이므로 정확히 이해하는 것이 중요합니다.

## 1. 공통 애노드 (Common Anode)

**"모든 LED의 (+)극이 하나로 묶여 전원(Vcc)에 연결된 방식"**

Basys3 보드의 7세그먼트가 이 방식입니다.

### 회로 구성

모든 LED의 **애노드(+)**가 내부적으로 함께 묶여 Vcc(+3.3V)와 같은 높은 전압에 연결됩니다. 각 LED의 **캐소드(-)**는 개별적으로 FPGA 핀에 연결되어 제어됩니다.

```
      [+3.3V 전원]
          |
          |
     (+) [LED] (-)
          |
          |
      [FPGA 제어 핀]
```

### 제어 원리 (Active-Low)

LED를 켜기 위해서는 전류가 흘러야 합니다. (+)극은 이미 +3.3V에 연결되어 있으므로, FPGA는 (-)극의 전압을 조절해야 합니다.

*   **LED 켜기**: FPGA가 제어 핀에 **`0` (LOW, 0V)** 신호를 출력합니다.
    *   **이유**: (+3.3V)와 (0V) 사이에 전압 차이가 발생하여, `Vcc -> LED -> FPGA` 방향으로 전류가 흘러 LED가 켜집니다.
*   **LED 끄기**: FPGA가 제어 핀에 **`1` (HIGH, +3.3V)** 신호를 출력합니다.
    *   **이유**: 회로 양단의 전압이 +3.3V로 동일해져 전압 차이가 없으므로 전류가 흐르지 않아 LED가 꺼집니다.

> **결론: 공통 애노드 방식은 `0`을 주어야 해당 LED가 활성화(ON)되는 Active-Low 방식입니다.**

#### ※ Basys3 보드의 `an` 신호
자릿수를 선택하는 `an` 신호 역시 Active-Low 입니다. 이는 FPGA 핀이 `0`이 될 때 켜지는 '전자 스위치(트랜지스터)'를 제어하여 해당 자릿수에 전원을 공급하기 때문입니다.

## 2. 공통 캐소드 (Common Cathode)

**"모든 LED의 (-)극이 하나로 묶여 그라운드(GND)에 연결된 방식"**

### 회로 구성

모든 LED의 **캐소드(-)**가 내부적으로 함께 묶여 GND(0V)와 같은 낮은 전압에 연결됩니다. 각 LED의 **애노드(+)**는 개별적으로 FPGA 핀에 연결되어 제어됩니다.

```
      [FPGA 제어 핀]
          |
          |
     (+) [LED] (-)
          |
          |
      [GND (0V)]
```

### 제어 원리 (Active-High)

(-)극은 이미 0V에 연결되어 있으므로, FPGA는 (+)극의 전압을 조절해야 합니다.

*   **LED 켜기**: FPGA가 제어 핀에 **`1` (HIGH, +3.3V)** 신호를 출력합니다.
    *   **이유**: (+3.3V)와 (0V) 사이에 전압 차이가 발생하여, `FPGA -> LED -> GND` 방향으로 전류가 흘러 LED가 켜집니다.
*   **LED 끄기**: FPGA가 제어 핀에 **`0` (LOW, 0V)** 신호를 출력합니다.
    *   **이유**: 회로 양단의 전압이 0V로 동일해져 전압 차이가 없으므로 전류가 흐르지 않아 LED가 꺼집니다.

> **결론: 공통 캐소드 방식은 `1`을 주어야 해당 LED가 활성화(ON)되는 Active-High 방식입니다.**

## 3. 요약 비교

| 특징 (Feature) | 공통 애노드 (Common Anode) | 공통 캐소드 (Common Cathode) |
| :--- | :--- | :--- |
| **공통 단자** | 모든 **(+)극** (Anode) | 모든 **(-)극** (Cathode) |
| **공통 단자 연결** | 전원 (Vcc, +3.3V) | 그라운드 (GND, 0V) |
| **제어 신호** | `0` (LOW) | `1` (HIGH) |
| **활성화 방식** | **Active-Low** | **Active-High** |
