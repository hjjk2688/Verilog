# 임베디드 시스템
## CPU, MCU, FPGA
CPU, MCU, FPGA는 전자 기기의 '두뇌'로 사용되는 핵심 부품들이며, **임베디드 시스템**은 이러한 부품들을 활용하여 만들어진 '특정 목적의 시스템'을 의미합니다. 

---

## 1. 임베디드 시스템 (Embedded System)

*   **정의:** 큰 기계나 시스템 내에서 특정 기능(예: 체온 측정, 모터 제어)을 수행하도록 **내장(embedded)된 컴퓨터 시스템**입니다. 하드웨어와 그를 제어하는 소프트웨어가 결합된 전체 시스템을 의미합니다.
*   **설명:**
    *   임베디드 시스템은 '부품'이 아닌 **'개념' 또는 '완성된 시스템'**에 가깝습니다.
    *   이 시스템의 두뇌 역할로 **CPU, MCU, FPGA** 등이 사용됩니다.
    *   예를 들어, 스마트 워치는 'MCU를 두뇌로 사용한 임베디드 시스템'이고, 자동차의 내비게이션은 '고성능 CPU를 두뇌로 사용한 임베디드 시스템'입니다.
*   **주요 용도:** 가전제품, 자동차, 항공기, 의료기기, 산업용 로봇 등 우리 주변의 거의 모든 전자 기기

---

## 2. CPU (Central Processing Unit, 중앙 처리 장치)

*   **정의:** 컴퓨터의 핵심 두뇌로, 복잡하고 다양한 연산을 매우 빠르게 처리하도록 설계된 범용 프로세서입니다.
*   **특징:**
    *   운영체제(Windows, Linux 등) 위에서 다양한 프로그램을 실행합니다.
    *   단독으로 동작하지 않으며, RAM, 저장장치 등 외부 부품이 반드시 필요합니다.
    *   성능이 가장 강력하지만, 전력 소모가 크고 가격이 비쌉니다.
*   **주요 용도:** PC, 노트북, 서버, 스마트폰 등 고성능 범용 컴퓨팅 기기

---

## 3. MCU (Microcontroller Unit, 마이크로컨트롤러)

*   **정의:** 하나의 칩(Chip)에 CPU 코어, 메모리(RAM, Flash), 입출력(I/O) 포트가 모두 통합된 '단일 칩 컴퓨터'입니다.
*   **특징:**
    *   칩 하나만으로 독립적인 시스템을 구성할 수 있어 회로가 단순해집니다.
    *   특정 기능(모터 제어, 센서 값 수집 등)을 반복적으로 수행하는 데 최적화되어 있습니다.
    *   전력 소모가 매우 적고 가격이 저렴하여 대량 생산에 유리합니다.
*   **주요 용도:** IoT 기기, 가전제품, 자동차 전장 부품, 로봇 등 각종 제어 장치

---

## 4. FPGA (Field-Programmable Gate Array)

*   **정의:** 개발자가 내부의 논리 회로를 직접 설계하고 프로그래밍할 수 있는 '텅 빈 캔버스' 같은 반도체입니다.
*   **특징:**
    *   하드웨어 기술 언어(HDL)를 사용하여 회로를 직접 설계하므로, 소프트웨어 방식보다 훨씬 빠른 속도를 낼 수 있습니다.
    *   수많은 작업을 동시에 처리하는 **병렬 처리**에 매우 강력한 성능을 보입니다.
    *   개발 난이도가 높고, MCU보다 비싸지만 원하는 기능의 맞춤형 하드웨어를 만들 수 있는 최고의 유연성을 제공합니다.
*   **주요 용도:** 고속 신호 처리, 5G/6G 통신 장비, 인공지능(AI) 가속기, 반도체 프로토타입 제작

---

## 부품별 한눈에 보는 비교표 (CPU vs MCU vs FPGA)

| 항목 | **CPU** | **MCU** | **FPGA** |
| :--- | :--- | :--- | :--- |
| **분류** | 중앙 처리 장치 | 단일 칩 컴퓨터 | 프로그래밍 가능 논리 회로 |
| **핵심 역할** | 범용 고속 연산 | 특정 목적의 제어 | 맞춤형 하드웨어 로직 구현 |
| **시스템 구성** | RAM, 저장장치 등 필요 | **단일 칩으로 동작 가능** | 주변 회로 필요 |
| **프로그래밍** | C++, Python 등 고급 언어 | C/C++, 어셈블리 | **Verilog, VHDL (하드웨어 언어)** |
| **성능** | 매우 높음 | 낮음 ~ 중간 | **병렬 처리에 극강** |
| **전력 소모** | 높음 | **매우 낮음** | 중간 ~ 높음 |
| **가격** | 높음 | **매우 저렴** | 중간 ~ 매우 높음 |

## 요약

*   **임베디드 시스템:** 특정 목적을 위해 만들어진 **완성형 제어 시스템**.
*   **CPU:** 똑똑하고 뭐든 잘하지만, 비싸고 전기를 많이 먹는 **만능 두뇌**.
*   **MCU:** 작고, 저렴하며, 한 가지 일만 효율적으로 하는 **전용 일꾼**.
*   **FPGA:** 개발자가 직접 회로를 만들어 원하는 대로 기능을 바꿀 수 있는 **하드웨어계의 레고(LEGO)**.

위 부품(CPU, MCU, FPGA)들을 사용하여 **임베디드 시스템**을 만듭니다.
